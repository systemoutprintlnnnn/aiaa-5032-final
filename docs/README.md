# Project Docs

This directory stores planning and implementation guidance for AI agents working on this project.

Use these docs to understand what to build and what not to build:

- `PLAN.md`: corrected architecture and phased implementation plan.
- `ARCHITECTURE.md`: current and target system architecture.
- `API_CONTRACT.md`: backend request/response contracts.
- `DATA_SOURCES.md`: data-source boundaries and license notes.
- `ROADMAP.md`: milestone sequence.

Root-level `AGENTS.md` remains the project instruction entry point. Detailed planning documents belong here.
