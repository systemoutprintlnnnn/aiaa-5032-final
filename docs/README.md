# Project Docs

This directory stores the current project documentation plus historical design records for AI agents working on this repository.

Use these docs to understand the completed codebase and its boundaries:

- `PLAN.md`: completed project plan and delivered scope.
- `ARCHITECTURE.md`: current system architecture and extension boundaries.
- `API_CONTRACT.md`: backend request/response contracts.
- `DATA_SOURCES.md`: data-source boundaries and license notes.
- `LOCAL_RUNBOOK.md`: local backend, frontend, Qdrant, and real RAG run steps.
- `ROADMAP.md`: completed milestone state.
- `superpowers/specs/` and `superpowers/plans/`: historical design and implementation plans used to build the current code. Treat them as provenance, not the current source of truth.

Root-level `AGENTS.md` remains the project instruction entry point. Detailed planning documents belong here.
